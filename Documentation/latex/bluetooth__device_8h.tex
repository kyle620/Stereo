\doxysubsection{Detailed Description}
This file holds information about remote bluetooth devices. 

\begin{DoxyAuthor}{Author}
Kyle Van Cleave 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
March 14, 2020
\end{DoxyDate}
This file keeps track of devices found during a bluetooth scan, and any device that connects to the raspberry pi. It can update the Bluetooth device properties Files that use this file are

\item \mbox{\hyperlink{bluez__adapter__api_8h}{bluez\+\_\+adapter\+\_\+api.\+h}} (adds devices found during scan)
\item \mbox{\hyperlink{bluez__device__api_8h}{bluez\+\_\+device\+\_\+api.\+h}} (adds devices when connections are made) 
\end{DoxyItemize}
